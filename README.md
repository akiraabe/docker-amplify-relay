# Amplifyによる駅伝アプリ

## このプロジェクトの目的

* FrontEndはReactで作る
* Amplifyを用いてCognito認証を組み込む
* AppSync + DynamoDBのバックエンドと連携する
* Amplify customを用いて、cdkとSNSの連携をする
